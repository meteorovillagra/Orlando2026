Cómo reservar para los 8 en una sola operación
⌄

1. Vinculación inicial: Una persona conecta las 3 habitaciones en la App como Family & Friends para que el sistema reconozca a los 8 perfiles.

2. Juegos en grupo (7:00 AM): Para las atracciones donde van los 8 juntos, un solo Admin reserva para todos a las 7:00 AM para asegurar exactamente el mismo horario.

3. Juegos divididos / opcionales: Si una atracción es solo para adultos o para un subgrupo, el Admin simplemente desmarca a quienes no suben, o cada habitación reserva lo suyo.

💡 Tip de Autonomía: Una vez hecha la reserva de las 7:00 AM, cada habitación puede cambiar la hora o cancelar su propio pase desde su propio celular de forma 100% independiente.
